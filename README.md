# IAM-Automation-Workflow

This project demonstrates an identity-driven automation process for provisioning:

- Active Directory security groups
- User-to-group assignments
- File system folders
- NTFS permissions

The automation is based on organizational hierarchy (Division, Department, Group, Team).

## Key Capabilities
- Dynamic group creation
- Automated access provisioning
- Identity-based authorization model
- Audit logging

## Technologies
- PowerShell
- Active Directory
- NTFS Permissions
